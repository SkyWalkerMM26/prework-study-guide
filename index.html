  • Meta elements contain informaton about the page that isn used by the browser. The user won't see information in the head or meta elements.
  • HMTL document elements: !Doctype, html, head, meta, title.
  • href attribute point to the URL for the link.

CSS

  • CSS BOX MODEL goes from Margin, Border, Padding, then to Content (Exterior to Interior)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are incline css, internal style sheet, and external css style sheet.
  • External style sheet is the most preferred because it allows develipers to keep all their Css rulls in a seperate file.

Git

  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "x" commit a message
  • git pull origin main update the local repository to match the repote repo.
